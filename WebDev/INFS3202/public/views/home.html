Introduction

Welcome to the Events Explorer web application. This is the home page for an individual project developed for the university course INFS3202. Using the MEAN development stack, and Angular Material UI Library.

To navigate the website, the left side panel is a navigation drawer. There is a webpage for help on FAQ. An option is also included to leave a review for the website.

Go to help section Application Features Explore, Save Events and Leave a Review

The main features of the application all revolve around Events, being able to search for and save events. After users have logged in, they can save and then later view their saved events.

In addition, users can also leave a review about the website and view reviews from other users.
